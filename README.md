# 今晚不回家 🍻 酒桌必玩

酒桌 / 聚会 / 宿舍 / 团建用的互动小游戏，手机打开即玩，无需注册。

## 玩法

- **转盘喝酒**：转盘随机指到一人，随机触发喝酒或惩罚。
- **真心话大冒险**：普通 / 刺激模式，随机出题。
- **谁是卧底**：选人数后发词，传手机轮流查看，描述后投票找卧底。
- **抽卡惩罚**：抽一张卡，喝酒 / 指定别人 / 免罚 / 反弹等，立即执行。

## 部署到自己的服务器

1. 把整个 `drink_game` 文件夹上传到服务器（如 Nginx、Apache 的网站目录）。
2. 确保服务器能访问：
   - `index.html`
   - `css/style.css`
   - `js/data.js`
   - `js/app.js`
3. 用浏览器访问对应网址即可（建议用 HTTPS，手机访问更稳）。

### 本地预览

```bash
# 用 Python 起一个简单 HTTP 服务（在项目根目录执行）
python -m http.server 6969
# 浏览器打开 http://localhost:6969
```

或用 VS Code 的 Live Server 打开 `index.html`。

## 技术说明

- 纯前端：HTML + CSS + JS，无后端、无数据库。
- 移动端优先布局，适合手机浏览器。
- 支持夜间模式（设置会保存在本地）。
- 题库在 `js/data.js`，可自行增删改。

## 文件结构

```
drink_game/
├── index.html      # 主页面，包含所有玩法视图
├── css/
│   └── style.css   # 样式（含夜间模式）
├── js/
│   ├── data.js     # 题库与配置（可编辑）
│   └── app.js      # 路由与各玩法逻辑

```
